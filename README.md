# 📚 Front Sistema Alunos

Frontend do sistema de cadastro de alunos do projeto social de vôlei **MatchPoint**.  
O objetivo do projeto é facilitar o gerenciamento dos alunos cadastrados, permitindo operações completas de CRUD e autenticação administrativa.

---

## 🚀 Funcionalidades

- ✅ Cadastro de alunos
- ✅ Listagem de alunos
- ✅ Edição de alunos
- ✅ Exclusão de alunos
- ✅ Sistema de login para administrador
- ✅ Navegação entre páginas
- ✅ Interface moderna e responsiva

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

- ⚛️ ReactJS
- 📘 TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS
- 🛣️ React Router DOM
- 🔥 Axios
- 📦 Node.js
